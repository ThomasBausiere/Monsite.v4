1.1 Les variables

Déclaration de variables

Les noms de variables peuvent contenir n'importe quels lettres, chiffres et autres mais ne peuvent pas commencer par un chiffre
Il existe trois mots clefs pour déclarer une variable:


  • let est le mot clef de déclaration de variable le plus conseillé. Je peux attribuer une valeur à une variable au moment de sa déclaration, ou plus tard.
    • let banane;

    Je peux déclarer plusieurs variables en même temps en les séparant d'une virgule.

    let a, b=2, c;
  • const permet de déclarer une constante
    • const cerise = 5;

    Une "constante" est une variable qui, une fois définie ne peut pas changer. On est dailleurs obligé de la définir au moment de la déclaration, contrairement à la variable.

    Par ailleurs, const utilise les mêmes régles de portée que "let".On ne peut changer la valeur d'une constante ni redéclarer une variable


  • var

    déclare une variable avec une plus grande portée.(voir sous chapitre suivant)

    var tomate=3;

La porté des variables

Lorsque l'on code, on va créer des "blocs" souvent représenté par des accolades "{}".

Que ce soit avec des conditions, des fonctions, ou même généralement en javascript, chaque bloc est séparé du reste.

C'est là qu'entre en jeu, la portée des variables.

var a une portée dites " globale", elle sera accessible et modifiable depuis bien plus d'endroit que le let, qui elle sera cloisoné dans son bloc.

let a été créé pour éviter les erreures de modification d'une variable par accident.

un let déclaré dans un bloc, n'existe que dans celui ci ou dans ses descendants.

Les tableaux

Un tableau est une liste d'éléments, il peut etre déclaré de deux façons différentes:

let a1=[5, "truc", true];
let a2=new Array(5, "truc", true);

Pour accéder à un element précis du tableau, il me placer des crochets " [ ] " après la variable dans lesquel j'indique l'index que je souhaite avoir

console.log(a1[1]);

.length permet d'indiquer la taille du tableau.

console.log(a1.length);

Pour obtenir le dernier élément d'un tableau de taille variable.

console.log(a1[a1.length-1]);

Depuis Ecmascript 2022, il est aussi possible de faire ceci:

console.log(a1.at(-1));

Cela fonctionne aussi avec un string:

console.log("salut".at(-1),"Salut"[2]);

quelques fonctions:

Pour ajouter un élément à la fin du tableau

a1.push("Bidule");

Pour retirer un élément à la fin du tableau

a1.pop();

retirer un élément au début du tableau

a1.shift();

Ajouter un élément au début du tableau:

a1.unshift(42);

Suprimer l'index du tableau et ce qui suit

a1.splice(1)

Supprime à partir de l'index indiqué, le nombre d'élément choisi.

a2.splice(1,1)

Supprimer à partir de l'index indiqué, le nombre d'élément choisi et les remplacer à partir de ce qui suit.

a2.splice(1, 1, "chausettes", "banane");

Insérer sans rien supprimer:

a2.splice(2, 0,"cerise");

sort(permet de trier par ordre alphabétique... même les nombres)

a3.sort();

Il est également possible d'ajouter un élément dans un tableau en lui indiqueant l'index a prendre.

a1[8] = "pomme";

On prefera dans ce cas lui dire:

a2[a2.length]="pomme";

Si on souhaite faire une copie d'un tableau:

a noter qu'une variable ou une constante de type tableau comme const i=[]; indique techniquement l'adresse d'un tableau mais n'est pas un tableau en lui même.

donc un simple let b= i; sera une copie parfaite et la modification de l'un modifiera l'autre (même les const)

Pour éviter cela, on décomposera le tableau grâce au "spread operator" représenté par des " ... ": comme suit:

a4=[...a2];

on peut aussi se servir du spread operator pour fusionner un tableau dans un nouveau tableau.

let a5 =["pizza", ...a4, 'pomme', "bannane"];

Sans le spread operator, on se retrouve avec un tableau multi dimensionnel ( un tableau dans un tableau)

let a6=["pizza", a4, "pomme", "banane"];

pour accèder à une valeur d'un tableau multi dimensionnel, on selectionnera d'aboord celui ci avant d'ajouter l'index voulu dans le second tableau;

let a7=[[[[[['coucou']]]]]];

Pour échanger la position de deux valeurs dans un tableau, on pourra évidement utiliser une variable temporaire.

console.log("a3", a3);
let tmp=a3[0];
a3[0]=a3[3];
a3[3]=tmp;
console.log("a3", a3);

mais avec JS, on peut faire ainsi:

[a3[0], a3[1]] = [a3[1], a3[0]];

Les objets

Les objets js ressemblent aux tableaux mais utilisent "{}" au lieu de "[]". et au lieu d'etre indexé numériquement, is sont indexés par des string qui servent de nomm de propriétés On pourra par exemple dire que la props "nom" vaudra "dupont".

let o1 ={nom:"dupont", age:45, loisir:["bowling", "mahjong"]};

Pour accèder à une propriéte d'un objet, je peux le faire comme avec un tableau

console.log(o1, o1["nom"], o1["age"], o1["loisir"]);

Mais les objets peuvent aussi être utilisé ainsi:

console.log(o1.nom, o1.age, o1.loisir);

Ici dans un premier cas, il va chercher la valeur v et dans le seconde la proprieté

let v="age";
console.log(o1[v], o1.v);

il faut donc utiliser la methode avec les "[]" lorsque la valeur que l'on souhaite est une variable.

Si on veut accèder à un tableau dans un objet, il suffit de le sélectionner.

console.log(o1.loisir[0], o1["loisir"][0])

On peut avoir autant d'objets dans un objet que voulu

let o2={vegetal:{legume:{haricot:{couleur:"vert"}}}};
console.log(o2.vegetal.legume.haricot.couleur);

Comme pour les tableaux, on ne peut pas copier sans modifier l'original

On devra aussi utiliser le spread operator:

let o4 ={...o1};
o4.signe="scorpion";
console.log(o1, o4);

on peut aussi se servir du spread operator pour transformer un tableau en objet

console.log(a1, {...a1});

Dans le cas d'une fusion d'objet, les propriétés uniques seront ajoutées et celles en commun, se verront remplacées par la dernière

let o5 ={...o4, ...o1};
console.log(o5);

Les Booleans

Les booleans n'acceptent que deux valeurs: "true" ou "false"

let b1= true, b2=false;

On peut faire apparaitre des booleans de nombreuses façons :

Du plus petit ou plus grand:

console.log("1 < 2:", 1 < 2);
console.log("1 > 2:", 1 > 2);

Du plus petit ou égal, plus grand ou égal

console.log("1 <= 2:", 1 <= 2); console.log("1 >= 2:", 1 >= 2);

Pour vérifier une égalité:

console.log("1 == 1:", 1 == 1);
console.log("1 == '1':", 1 == '1');

Pour vérifier une égalité stricte:

console.log("1 == 1:", 1 === 1);
console.log("1 == '1':", 1 === '1');

Pour vérifier la différence ou la différence stricte:

console.log("1 != '1':", 1 != '1');
console.log("1 !== '1':", 1 !== '1');

On peut inverser un résultat en le faisant précéder d'un "!";

console.log("b1:", b1, "b2:", b2);
console.log("!b1:", !b1, "!b2:", !b2);

On peut regrouper les booleans avec les opérateurs "et" et "ou" representé par les symboles " &&" et "||" :

console.log(true && true);
console.log(true && false);
console.log(false && true);
console.log(false && false);

console.log(true || true);
console.log(true || false);
console.log(false || true);
console.log(false || false);

On peut ajouter des parenthèses à tout cela

console.log(true&&(false||true));

On peut "court-circuiter" un code avec ces comparateurs:

Dans le cas d'un &&, si le premier est faux, le second ne sera pas vérifier.

console.log(true && ++nb, nb);
console.log(false && ++nb, nb);

Dans le cas d'un ||, si le premier es vrai, le second ne sera pas vérifié.

console.log(false || ++nb, nb);
console.log(true || ++nb, nb);
